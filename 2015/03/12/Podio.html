Trello
A fast and visual collaboration task manager where projects and tasks are organized by cards and boards. This “kanban” view has raised extreme critics, users either like it a lot or consider it off-putting.
Wunderlist
A simple and collaborative list manager you can easily share with your friends, colleagues and family. Its native app works smoothly and synchronizes seamlessly across almost all devices. Ideal for users who switches their workspace from PC, tablet and mobiles all the time.
Quire
Fast, intuitive and easy to use. 3-section panel is similar to Asana but provides a much cleaner and simpler UI with an unique tree structure which present users their project’s big picture at first view.
Basecamp

A simple, and quick to set up task management tool where each project is represented by a “card” with a todo lists offering the most basic features. Inexpensive but provides commercial version only.

Redbooth
Enterprise like UI offering more advanced features like HD meeting, time tracking and reporting. Assigned subtasks automatically becomes another first level task which may cause some confusion for users.
Asana

A complete project management tool and workspace keeping teams organized, focused and connected. Performance may be an issue to compensate for its many features.

Producteev
A well balanced task manager offering all features you general need for a task manager with an adequate UI and seamless email integration. However, “activity feed” of all team members may be excessive for some users.
Podio

Offers probably the most complete set of features and an unique add-on application database to customize your workspace for any industry. However, the complicated UI and UX makes it hard for users to get started.

Flow

Offers a clean UI and basic features of a task manager while seamlessly synchronizes across PC, tablet and mobiles. Some of its automatic categories can be very useful for managerial level personnels.

Google Tasks

Seamlessly integrated with Gmail and Google Calendar, Google Tasks is succinct and simple to use for keeping personal to-do lists.

Taskulu
Following Trello’s footsteps, Taskulu is another kanban type of task manager but with more advanced features such as timelogs and project chatroom. Its UI design is also cleaner and more modern than Trello.
MS Project

An end-to-end project management tool used from planning, budgeting to allocating resources and task scheduling. It is ideal for tracking large scale projects but may be too heavy for a small team of 10 or so.
